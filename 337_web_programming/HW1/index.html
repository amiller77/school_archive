An Intro to Web Programming

What is HTML?

HTML is considered a “markup” language, rather than a “programming” language.

Today, HTML is used only to declare “what” the different components of a website are (link, paragraph, etc.) rather than to format or stylize in a detailed sense – that is left to CSS for clarity and efficiency.

HTML Syntax

The syntax is organized by “tags,” which are enclose in brackets.

“Tags” allow you to customize the appearance of a web page and its text, add to the header or body, and begin and end your HTML code.

Any tags which are nested must be fully encapsulated within the other tag; no “overlapping” tags.

What is CSS?

Whereas HTML is for content, CSS allows you to customize style, format, and appearance with an efficiency, ease, and clarity that cannot be reached by trying to do this functionality in HTML.

CSS Syntax

  • A CSS file consists of one or more rules
  • A rule’s “selector” specifies the HTML element to which style properties should be applied.
  • Choosing * as a selector selects all elements
  • You can also choose multiple selectors to modify, or modify the same one several times in different groupings.

Some Background Properties

  1. background: shorthand to set all background properties
  2. background-color: fill background with color
  3. background-image: place image in background
  4. background-position: placement of background image in element
  5. background-repeat: whether background image should be repeated
  6. background-attachment: if background image scrolls with page
